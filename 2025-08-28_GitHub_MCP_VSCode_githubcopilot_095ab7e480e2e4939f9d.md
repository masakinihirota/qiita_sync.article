## 推奨設定

**個人開発時（約15-20個のツール）：**
- Repository基本操作
- ファイル操作
- Issue/PR基本機能
- Branch操作

**組織開発時（追加で10-15個）：**
- チーム管理機能
- 権限管理
- 組織設定

これにより、91個から30-35個程度まで削減でき、git（約10個）とFigma（約10-15個）と合わせても制限内に収まるはずです。
